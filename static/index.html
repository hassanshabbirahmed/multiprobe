Multiprobe Endpoints

  • /tag: /tag - Displays the application tag.
  • /hostname: /hostname - Displays the server hostname.
  • /both: /both - Displays both tag and hostname.
  • /primetime: /primetime - Calculates prime numbers.

    Note: This endpoint can be slow to respond as it performs a significant computation.

  • /metrics: /metrics - Exposes Prometheus metrics.
